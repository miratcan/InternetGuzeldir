Akımlar

  • Üst Klasör
  • The 1Mb Club

    250Kb club gibi, ancak biraz daha kepçeyi bol tutup üst limiti 1Mb ile sınırlamışlar.

  • TextOnly Websites

    Basit, text odaklı web sitelerinin bir dizini. Bu siteler hızlı yüklenmeleri, kolayca gezilmeleri, az pil tüketmeleri, daha okunur olmaları ve birçok web sitesinde var olan olağan saçmalıklardan yoksun olmalarıyla takdir görüyorlar.

  • The 250Kb Club

    Bir eposta listesini görüntülemek için 8mb javascript, 6mb asset indirmenin yanlış olduğunu düşünenlerin klübü. İçerde 250Kb altında indirme gerektiren web sitelerinin listesi var.

  • The NoCSS Club

    CSS kullanmayan web siteleri klübü.

  • The NoJS Club

    Tüm web sitelerinin güzel ve kullanışlı olması için Javascript'e ihtiyacı olmadığını savunan bir akım. İçerde bu akıma uyan, Javascript içermeyen web sitelerinin listesi var.
