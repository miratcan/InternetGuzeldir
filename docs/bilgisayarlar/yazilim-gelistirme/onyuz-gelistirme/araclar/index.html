Araçlar

  • Sizzy Browser - Önyüz geliştiricişleri için özelleştirilmiş internet tarayıcısı. Yaptığınız bir değişikliği aynı anda birden fazla cihaz çözünürlüğü üzerinde gösterebilmek gibi faydalı ve hızlandırıcı özellikleri var.
  • Astro Build - React, vue gibi geliştirme çatılarıyla geliştirilmiş sayfalarını, mümkün olduğunca Javascript'den arındırarak statik web sayfaları haline getirmeyi, ve bu sayede performans artışı sağlamayı hedefleyen bir araç.
  • Get Waves - CSS ile kod yazmadan dalga formu yapmanızı sağlayan basit uygulama.
  • Blob Maker - Css ile kod yazmadan damla formu yapmanızı sağlayan basit uygulama.
  • Animista - Kod yazmadan CSS animasyon tasarımları yapmanızı sağlayan online uygulama.
